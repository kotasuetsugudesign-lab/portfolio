LUMERA

Role：Design / Coding ｜ Period：4 days

ブランドの世界観を軸に設計した、
アパレル訴求型ランディングページ。

サイトを見る

PC / タブレット / スマートフォン完全対応。
scroll-snapによる横スクロール設計と
IntersectionObserverによるスクロール演出を実装。

プロジェクト概要

架空のアパレルブランドを想定した商品訴求型LP。
20代後半〜30代のシンプル志向の女性をターゲットに、世界観と購買導線を両立する設計を行いました。

設計意図と実装戦略

  • Hero → Concept → Collection → Lookbook → Philosophy のブランド導線設計
  • 横スクロール商品グリッドに scroll-snap を実装
  • IntersectionObserver によるスクロールアニメーション
  • requestAnimationFrame を用いたヘッダースクロール最適化
  • aria属性付きハンバーガーメニュー＋ESCキー対応
  • 画像の lazy loading による表示最適化

使用技術

HTML CSS JavaScript Responsive Design IntersectionObserver scroll-snap requestAnimationFrame Accessibility (ARIA)
